The API

What you get in one call.

Full factor attribution, directional signal, confidence score, and expected alpha — structured and ready to plug into your own stack.

GET /v1/signals?ticker=AAPL&lookback=30d
ticker string e.g. AAPL, BTC, SPY
lookback string 7d · 30d · 90d · 1y
factors array? filter factor set
confidence_min float? 0.0 – 1.0 threshold
