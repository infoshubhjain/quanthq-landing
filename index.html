The API

What you get in one call.

Full factor attribution, directional signal, confidence score, and expected alpha — structured JSON ready for your own stack.

GET /v1/signals?ticker=AAPL&lookback=30d

lookbackstring7d · 30d · 90d · 1y
factorsarray?filter factor set
confidence_minfloat?0.0 – 1.0 threshold
